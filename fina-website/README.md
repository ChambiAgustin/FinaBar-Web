# Fina Cocktail Bar Website 🍸

Sitio web oficial de Fina Cocktail Bar, Río Cuarto.
Desarrollado con HTML, CSS (Vanilla) y JavaScript.

## Características
- **Carta Digital Interactiva:** Buscador en tiempo real y filtros por categoría.
- **Diseño Responsive:** Optimizado para móviles (iOS/Android) y escritorio.
- **Modo Oscuro/Neon:** Estética inmersiva alineada con la marca.
- **Lightbox:** Galería de imágenes para los cocteles y platos.
- **Quiz Interactivo:** Juego para descubrir tu trago ideal.


## Cómo ver el sitio
Simplemente abre `index.html` en tu navegador.

## Despliegue
Este proyecto está listo para ser desplegado en Vercel, Netlify o GitHub Pages.
